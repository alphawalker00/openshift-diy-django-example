About

A Django 1.4 example project for deployment on http://openshift.redhat.com "Do It Yourself" application type

Features

  • Django 1.4
  • 'django.contrib.staticfiles' support
  • Pip
  • South
  • MongoDB (caching and Celery support ~ work in progress)
  • Gunicorn
  • Postgresql

Notes

The end goal is to manage the application with Supervisord, including the gunicorn and celeryd processes but that is not currently posssible due to the lack of shared memory support
